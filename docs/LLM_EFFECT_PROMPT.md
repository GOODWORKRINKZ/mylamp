# LLM Effect Prompt

Этот файл нужен для быстрого копирования prompt-а, который просит LLM сгенерировать эффект строго в рамках DSL проекта mylamp.

Когда использовать:
- когда хочешь быстро получить новый эффект от LLM;
- когда нужно напомнить модели ограничения языка;
- когда важно, чтобы ответ был только DSL-кодом без лишнего текста.

Как использовать:
1. Скопируй prompt целиком.
2. Вставь его в LLM.
3. Подмени строку `ЗАДАЧА:` своим описанием эффекта.
4. Возьми полученный DSL и вставь его в редактор mylamp.

## Copy-Paste Prompt

```text
Ты генерируешь эффект для pixel lamp проекта mylamp.

Нужно вернуть ТОЛЬКО DSL-код эффекта, без markdown, без пояснений, без комментариев.

Полотно лампы:
- логический размер 32x16
- эффект должен быть рассчитан на этот размер

Разрешённый синтаксис DSL:
- effect "name"
- sprite <name> { bitmap """ ... """ }
- layer <name> { ... }

Разрешённые поля внутри layer:
- use <spriteName>
- color rgb(...)
- color hsv(...)
- x = <expression>
- y = <expression>
- scale = <expression>
- visible = <expression>

Разрешённые переменные:
- t
- dt
- x
- y
- nx
- ny

Разрешённые функции:
- sin(...)
- cos(...)
- abs(...)
- min(...)
- max(...)
- clamp(...)
- temp()
- humidity()
- rgb(...)
- hsv(...)

Ограничения:
- не используй циклы
- не используй if
- не используй function
- не используй let/var/const
- не придумывай новые ключевые слова
- не придумывай новые поля layer кроме use/x/y/scale/color/visible
- не используй комментарии
- не возвращай JSON
- не возвращай пояснения
- не используй markdown code fences

Требования к результату:
- эффект должен начинаться с effect "..."
- должен быть хотя бы один sprite
- должен быть хотя бы один layer
- код должен быть компактным и валидным по описанному DSL
- если нужна простая форма, используй bitmap из символов # и .

ЗАДАЧА:
Сгенерируй эффект для лампы: <сюда вставь описание эффекта>
```

## Пример пользовательского запроса

```text
Сгенерируй эффект для лампы: тёплое сердечко, которое медленно плывёт слева направо, слегка дышит и мягко меняет яркость.
```

## Пример ожидаемого ответа

```text
effect "warm_heart"

sprite heart {
  bitmap """
  .##.##.
  #######
  #######
  .#####.
  ..###..
  ...#...
  """
}

layer love {
  use heart
  color rgb(255, 80 + 30 * sin(t * 2), 90)
  x = 3 + t * 2
  y = 4 + sin(t * 1.5) * 2
  scale = 1 + abs(sin(t * 2)) * 0.4
  visible = 1
}
```
