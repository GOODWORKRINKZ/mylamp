# LLM Prompt File Design

**Date:** 2026-03-23

## Goal

Вынести copy-paste prompt для генерации DSL-эффектов в отдельный markdown-файл, чтобы:
- README оставался читаемым;
- prompt было удобно открывать и копировать отдельно;
- правила для LLM можно было развивать независимо от общей документации.

## Decision

Использовать отдельный файл `docs/LLM_EFFECT_PROMPT.md`.

README должен:
- сохранить раздел про рекомендации работы с LLM;
- кратко объяснять, зачем нужен отдельный prompt-файл;
- давать прямую ссылку на `docs/LLM_EFFECT_PROMPT.md`;
- оставить короткий пример использования.

## Content of the prompt file

Файл должен содержать:
- краткое описание назначения;
- правила использования;
- один основной copy-paste prompt;
- пример пользовательского запроса;
- пример ожидаемого DSL-ответа.

## Constraints

- Не дублировать длинный prompt одновременно и в README, и в отдельном файле.
- Не заявлять поддержку синтаксиса, которого нет в текущем DSL-контракте.
- Сохранять русскоязычную документацию и английский-core синтаксис DSL.
