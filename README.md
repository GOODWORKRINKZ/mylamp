# mylamp

Пиксельная лампа на ESP32-C3 Super Mini с двумя матрицами WS2812B 16x16, свернутыми в цилиндр.

## Что строим

Первая версия проекта собирается как production-прошивка на PlatformIO/C++.

Цели v1:
- эффекты на цилиндрической лампе;
- часы поверх эффектов;
- температура и влажность с AHT30;
- управление по WiFi через веб-интерфейс;
- OTA-обновления через GitHub Releases;
- подготовка к dev/stable каналам по образцу microbbox.

Что не является базой v1:
- MicroPython как основная прошивка;
- музыкальный эквалайзер как обязательная часть первого релиза.

## Почему не MicroPython в основе

Для этой лампы нужно совместить несколько тяжелых подсистем: рендер адресных матриц, веб-управление, OTA, хранение настроек, overlay часов и сенсоров, а позже еще и аудио-реактивный режим. Для ESP32-C3 это лучше укладывается в C++/Arduino через PlatformIO. Идею "вайбкодинга" имеет смысл вернуть позже как отдельный dev-режим или как скриптуемые параметры эффектов, но не как основу production-сборки.

## Базовая архитектура

Логическая модель экрана:
- единый canvas 32x16;
- wrap по X для цилиндра;
- две физические панели 16x16;
- serpentine mapping на уровне layout-модуля.

Слои системы:
- device config и build info;
- matrix layout и framebuffer;
- effect engine;
- overlays для часов и датчиков;
- WiFi/settings/web API;
- OTA и release/versioning.

## Стартовый каркас репозитория

- `platformio.ini` — базовая конфигурация сборки;
- `include/AppConfig.h` — аппаратные и логические константы;
- `include/BuildInfo.h` — версия, канал и идентификатор платы;
- `include/MatrixLayout.h` — интерфейс маппинга логических координат;
- `src/MatrixLayout.cpp` — реализация маппинга 32x16 -> 2x16x16;
- `src/main.cpp` — минимальная bootstrap-прошивка;
- `.agents/skills/` — generic skills, перенесенные из rob_box_project.

## Репозиторные решения

- generic skills копируются в `.agents/skills` и живут внутри проекта;
- robot-specific skills из исходного проекта не переносятся без адаптации;
- release engineering будет строиться по паттерну microbbox: draft-first release, stable/dev каналы и OTA через GitHub Releases.

## Важные аппаратные замечания

- питание матриц должно быть внешним, не от USB ESP32-C3;
- яркость по умолчанию будет ограничена с учетом тока и температуры;
- pinout пока стартовый и может быть скорректирован после финального выбора платы и разводки.
