# Моя Лампа

## What This Is

Умная LED-лампа на ESP32-C3 с цилиндрической матрицей 32×16 (окружность × высота), веб-интерфейсом управления, DSL для живого кодирования эффектов (Lux) и OTA-обновлениями. Устройство само раздаёт Wi-Fi или подключается к домашней сети, показывает эффекты, часы, и позволяет программировать новые визуализации через браузер.

## Core Value

Живое программирование света — пользователь пишет код эффекта в браузере и мгновенно видит его на LED-матрице лампы.

## Context

- **Железо:** ESP32-C3 Super Mini, WS2812B LED-матрица, AHT30 датчик
- **Матрица:** физически свёрнута в цилиндр: 32 LED по окружности, 16 рядов по высоте (всего 512 LED)
- **Текущая проблема:** `MatrixLayout` реализован как плоская панель с серпантинной развёрткой, не соответствует реальной геометрии цилиндра
- **NTP:** в коде есть `TimePlanner` и `TimeRuntimeService`, но синхронизация не работает — часы не получают время
- **Язык Lux:** DSL для описания эффектов (арифметика, тригонометрия, переменные, условия, циклы `for`). Может потребоваться расширение для поддержки спрайтов и анимаций
- **Фронтенд:** один большой файл `main.ts` (2197 строк), требуется разбиение на модули (в планах, не в этом milestone)

## Constraints

- **Hardware**: ESP32-C3 (400 KB SRAM, 4 MB Flash) — ограниченная память для DSL-программ и эффектов
- **Performance**: LED-матрица 512 пикселей, частота кадров зависит от сложности DSL-эффекта
- **Display geometry**: цилиндр — координаты непрерывны по окружности (wraparound), дискретны по высоте
- **Language**: Lux DSL — нужно сохранить обратную совместимость существующих эффектов при расширении
- **OTA**: размер прошивки ~1.5 MB, должен помещаться в OTA-партицию
