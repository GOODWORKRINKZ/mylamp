# Requirements: Моя Лампа

**Defined:** 2026-06-02
**Core Value:** Живое программирование света — пользователь пишет код эффекта в браузере и мгновенно видит его на LED-матрице лампы.

## v1 Requirements

### NTP — Синхронизация времени

- [x] **NTP-01**: Устройство получает точное время по NTP после подключения к Wi-Fi (клиент или AP с интернетом)
- [x] **NTP-02**: `TimeRuntimeService` корректно синхронизирует время и предоставляет его другим сервисам

### CLOCK — Часы на матрице

- [ ] **CLOCK-01**: Часы отображаются как оверлей на цилиндрической LED-матрице
- [ ] **CLOCK-02**: Формат часов настраиваемый (12h/24h) через веб-интерфейс
- [ ] **CLOCK-03**: Оверлей часов не конфликтует с активными эффектами (корректное наложение)

### CYL — Цилиндрическая геометрия

- [x] **CYL-01**: `MatrixLayout` поддерживает цилиндрическую геометрию: координаты (angle, row) → физический индекс LED
- [x] **CYL-02**: Горизонтальная координата зациклена (wraparound): пиксель на угле 0° соседствует с пикселем на угле 360°
- [x] **CYL-03**: `FrameBuffer` адаптирован: ширина = окружность (32), высота = ряды (16)
- [x] **CYL-04**: Функции отрисовки (`drawPixel`, `fillRect`, `drawLine` и др.) учитывают wraparound по горизонтали
- [x] **CYL-05**: Все существующие скомпилированные эффекты корректно работают на цилиндре
- [x] **CYL-06**: DSL Lux-эффекты корректно рендерятся в цилиндрической системе координат

### PERF — Баги и производительность

- [ ] **PERF-01**: Устранено мерцание и визуальные артефакты на LED-матрице
- [ ] **PERF-02**: Стабильный фреймрейт рендеринга, без заметных просадок
- [ ] **PERF-03**: Убран дебаг-хартбит (чередование boot-solid / debug-columns) в релизных сборках

### DSL — Примеры и язык

- [ ] **DSL-01**: Добавлены демо-эффекты с анимацией спрайтов на цилиндре (например, Марио бегает по кругу)
- [ ] **DSL-02**: При необходимости расширен язык Lux для поддержки спрайтов, кастомных шрифтов или массивов пикселей
- [ ] **DSL-03**: Демо-эффекты доступны как пресеты в веб-интерфейсе

## Out of Scope

| Feature | Reason |
|---------|--------|
| Мобильное приложение | Веб-интерфейса достаточно |
| Голосовое управление | Отдельная большая тема |
| Интеграция с умным домом (Matter, HomeKit) | Слишком большой объём для v1 |
| Аутентификация API | Пока доверяем локальной сети |
| Разбиение main.ts на модули | Не в этом milestone |
| RTC / сохранение времени между перезагрузками | ESP32-C3 не имеет аппаратного RTC, будем синхронизировать при каждом подключении |
